Setup Instructions
  1. Go to Shopify Partners Dashboard
  2. Click Apps → Create app → Custom app
  3. Fill in:
    App name: Returnfy
    App URL:
    https://returnfy.me
  4. In Configuration → Allowed redirection URL(s):
    IMPORTANT: Add BOTH URLs separated by comma:
    https://returnfy.me/api/shopify-callback,https://returnfy.vercel.app/api/shopify-callback
  5. In Configuration → App setup:
    Add scopes: read_orders, read_products, read_customers
  6. IMPORTANT: Go to App extensions and DISABLE "App embed". We don't need theme extensions.
  7. Click Create app version → Release
  8. Install the app on your test store
  9. Copy API key (Client ID) and API secret key (Client Secret) from Settings and fill the form below
